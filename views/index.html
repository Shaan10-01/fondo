What is this site all about?
As of now, this website is intended to provide personal and customizable backgrounds for various other websites along with chrome extension. So far it works best with VIT website.
In future, various other features such as confessions will be added. Method provided below is the best way until we provide the extension to chrome web store. Sorry!
Note: This website is in beta mode.


To get this working,

  • Download the chrome extension from store through this link.

  • Open chrome extensions tab (chrome://extensions) and enable developer mode provided on right side.

  • Unzip the downloaded file and click on 'load unpacked' and select the unzipped folder and there you go...

  • Create a new ID from create button.

  • Search for the ID in search bar and enter the password to set as profile.

  • Click on profile to see your ID as profile and play around.

  • Addition of extension is just one time process and all updates are provided directly.

  • Note that passowrd on ID doesn't protects the images and just covers the Only me images from showing them unnecessarily.

  • After searching an ID, if the ID has no passowrd then just click on passowrd input box and hit enter.
