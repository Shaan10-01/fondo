What is this site all about?
As of now, this website is intended to provide personal and customizable backgounds for various other websites along with chrome extension. So far it works best with VIT website.
In future, various other features such as confessions will be added.
Note: This website is in beta mode.

To get this working,

  • Add the chrome extension from store through this link.
  • Create a new ID from create button.
  • Search for the ID in search bar and enter the password to set as profile.
  • Click on profile to see your ID as profile and play around.
